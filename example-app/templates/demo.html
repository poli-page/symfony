Render 01 · instant

Three SDK entry points for rendering the getting-started/welcome template: binary PDF (render->pdf), streamed PDF (render->pdfStream), and HTML preview (render->preview).

Press a button to render.

Documents 02 · store & retrieve

Persist a render (render->document), then fetch it back, preview it, list thumbnails, or soft-delete it. The four right-hand buttons unlock after a document exists.

Error handling 03 · typed exceptions

Sends a deliberately malformed version string to trigger INVALID_VERSION_FORMAT. The bundle's typed exceptions propagate the API code, message, and request-id straight through.

outputidle
Press the button to see the error payload.

Command line 04 · console

These two commands prove the bundle wires through to Symfony Console. Copy and run in a terminal alongside this server.

$php bin/console app:demo:render-to-file
$php bin/console poli-page:render --project=getting-started \ --template=welcome --template-version=1.0.0 --output=/tmp/smoke.pdf
